## Этап 3: Система анкет и ранжирования

**Сроки**: 3-4 недели

### Задачи

- [ ] Разработка CRUD для анкет
- [ ] Реализация алгоритма ранжирования (минимум один пункт из каждого уровня)
- [ ] Внедрение системы кэширования анкет в Redis
- [ ] Интеграция с сервисом бота

### Подзадачи алгоритма ранжирования

#### Уровень 1: Первичный рейтинг
- [ ] Формирование на основе данных из анкеты (возраст, пол, интересы, география)
- [ ] Учёт полноты заполнения анкеты и количества фото
- [ ] Включение первичных предпочтений пользователя

#### Уровень 2: Поведенческий рейтинг
- [ ] Подсчёт количества лайков
- [ ] Расчёт соотношения лайков и пропусков
- [ ] Определение частоты мэтчей
- [ ] Учёт частоты инициирования диалогов

#### Уровень 3: Комбинированный рейтинг
- [ ] Интеграция первичного и поведенческого рейтингов
- [ ] Добавление бонусных баллов (рефералы)

### Оценка

| Пункт | Баллы |
|-------|-------|
| Один пункт из каждого уровня | 1 |
| Два пункта из каждого уровня | 2 |
| Все пункты из каждого уровня | 3 |

## Рекомендации

1. **Приоритизация**: Сначала реализуйте базовую функциональность, затем добавляйте дополнительные технологии
2. **Обоснование**: Каждая технология должна быть обоснована — она должна помогать, а не усложнять
3. **Тестирование**: Не забывайте про ручное и автоматическое тестирование
4. **Документация**: Документируйте решения по ходу разработки
