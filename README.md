# Dating Bot

Telegram-бот для знакомств с системой рейтингов анкет пользователей.

## Описание

Dating Bot — это приложение для знакомств, реализованное в виде Telegram-бота. Пользователи могут создавать анкеты, просматривать анкеты других пользователей, ставить лайки и общаться после взаимного совпадения (мэтча).

## Возможности

- Регистрация пользователей через Telegram
- Создание и редактирование анкет
- Система рейтинга анкет (3 уровня)
- Кэширование анкет в Redis
- Асинхронные задачи через Celery
- Потоковая обработка событий через RabbitMQ
- Загрузка фотографий в S3-хранилище (Minio)
- Логирование и метрики

## Система рейтинга

### Уровень 1: Первичный рейтинг
- Формируется на основе данных из анкеты (возраст, пол, интересы, география)
- Учитывает полноту заполнения анкеты и количество фото
- Включает первичные предпочтения пользователя

### Уровень 2: Поведенческий рейтинг
- Количество лайков анкеты
- Соотношение лайков и пропусков
- Частота взаимных лайков (мэтчей)
- Частота инициирования диалогов после мэтча
- Временные параметры (активность в определённое время суток)

### Уровень 3: Комбинированный рейтинг
- Интегрирует первичный и поведенческий рейтинги по весовой модели
- Учитывает дополнительные факторы (реферальная система)

## Технологии

- **Backend**: Python (FastAPI)
- **База данных**: PostgreSQL
- **Кэш**: Redis
- **Очереди**: RabbitMQ
- **Асинхронные задачи**: Celery
- **Хранилище файлов**: Minio (S3)
- **Бот**: Telegram Bot API
